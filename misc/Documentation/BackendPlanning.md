# Backend Planning Document

## Student Functionality

### Login Page
- **Purpose:** Allows students to authenticate and access their account.
- **Features:** 
  - Username/password authentication.
  - Secure session management.

### Search Page
- **Purpose:** Facilitates students in searching for research opportunities based on specific criteria.
- **Features:** 
  - Search by subject.
  - Filter by time/duration.
  - Display research descriptions.
  - Wishlist functionality.
  - View applied applications.

## Professor Functionality

### Login Page
- **Purpose:** Enables professors to authenticate and access their account.
- **Features:** 
  - Authentication with secure session management.

### Add/Remove Option
- **Purpose:** Grants professors control over research opportunities they offer.
- **Features:** 
  - Ability to add new research projects.
  - Option to remove existing projects.

### Received Applications
- **Purpose:** Provides professors with a consolidated view of applications submitted by students.
- **Features:** 
  - Centralized list of received applications.
  - Detailed information on each application.
